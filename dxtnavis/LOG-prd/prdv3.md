1. 전체 속성 csv로 저장할 때, 계측 구조 내보내기 처럼, 계층의 위아래를 확인할 수 있도록 내보내도록 하자.

2. 검색 세트 생성 (속성 기반)에서 사용자가 어떤 상호작용을 해야 data grid에 속성값들이 나타나는지 직관적으로 구현할 필요가 있어. 예를 들어서 1단계부터 7단계 까지 있는 항목에서, 1단계 항목을 선택된 객체로 지정한 경우, '선택된 객체의 속성'이 표시되는 공간에는 해당 객체와 그 객체의 계층 구조를 보여줬으면 좋겠어. UI 중간에 view를 하나 더 만들어서 왼쪽의 객체 계층 구조에서 몇 개 선택을 하면 중간의 view에서 해당 객체가 가지고 있는 속성값들이 등장해야해. 여기서 속성들을 선택할 수 있도록 구성하고, 이 선택된 속성들은 오른쪽 View에서 검색 세트 생성을 진행할 수 있도록 구성했으면 해.
